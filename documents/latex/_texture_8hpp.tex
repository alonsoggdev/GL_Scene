\doxysubsection{Detailed Description}
Clase para gestionar las texturas en Open\+GL. 

La clase {\ttfamily \doxylink{class_texture}{Texture}} permite la carga, enlace y liberación de texturas en Open\+GL. Se utiliza para manejar imágenes que se aplican a los objetos 3D en la escena, permitiendo efectos visuales como color, relieve, etc.
